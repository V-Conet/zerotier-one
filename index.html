ZeroTierOne for Alpine Linux

Usage

  1. Add the key 
    wget https://mirror.vconet.top/admin@vconet.top-69785d11.rsa.pub -O /etc/apk/keys/admin@vconet.top-69785d11.rsa.pub
  2. Add the mirror, please replace edge with your version if needed (e.g. v3.23) 
    echo "https://mirror.vconet.top/pkgs/edge/" | tee -a /etc/apk/repositories
  3. Enjoy it 
    apk update && apk add zerotier-one zerotier-one-openrc

Currently supports Alpine Edge, v3.23, v3.22, v3.21 on x86_64 x86 aarch64 loongarch64
